  Численное моделирование движения заряженной частицы в скрещенных электрическом и магнитном полях.
Рассматривается частица q = 1, m = 1, под действием силы Лоренца. Компоненты электричесокго поля Ex и Ey изменяются по гармоническому закону Ex = E0 * cos(omega * t), Ey = -E0 * sin(omega * t), где E0 = 0.5 - амплитуда, omega - частота. Магнитное поле постоянно и направлено вдоль оси Z, Bz = B0, где B0 = 1.0 - амплитуда магнитного поля. Параметры системы задаются в структуре SystemParams: 
 Ex = params.E0 * std::cos(params.omega * t); 
 Ey = -params.E0 * std::sin(params.omega * t); 
 Ez = 0.0; 
 Bx = 0.0; 
 By = 0.0; 
 Bz = params.B0;
Начальные условия заданы в InitialConditions (1,0,0), vx = 0.0, vy = 0.5, vz = 0.2, для отсутствия избыточных начальных осцилляций.
Итоговая система дифференциальных уравнений для системы: 
 dx/dt = vx, 
 dy/dt = vy, 
 dz/dt = vz, 
 dvx/dt = Ex + (vyBz - vzBy), 
 dvy/dt = Ey + (vzBx - vxBz), 
 dvz/dt = Ez + (vxBy - vyBx).

Реализовано 5 методов численного интергирования: rk4, rk3(8), rk5(6), dopri5, dopri835.

RK4 и RK3/8 являются фиксированными. RK4 использует коэффициенты 1/6, 1/3, 1/3, 1/6, а RK3/8 коэффициенты 1/8, 3/8, 3/8, 1/8.

RK5(6) адаптивный метод с контролем шага на основе сравнения решений пятого и шестого порядков, реализует схему Фельберга. Коэффициенты взяты из классического набора для пары 5(6).

Dopri5 с адапитвным шагом, метод пятого порядка с встроенным контролем ошибки четвертого порядка, используется семь стадий, оценка ошибки основана на разности между двумя вложенными формулами. Коэффициенты метода соответствуют классическому набору.

DOP853 используется как эталон для последующего сравнения методов, метод восьмого порядка с вложенной формулой третьего порядка для оценки ошибки.

Результаты представлены в виде девяти приложенных графиков, и численного сравнения ошибок.
